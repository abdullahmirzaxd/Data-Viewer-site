# Ratio Analysis Web App

This is a simple web app built using **Streamlit** that displays data from an Excel file (`ratio_analysis.xlsx`). The dataset is displayed in a tabular format using the `st.dataframe` function.

## Features:
- Reads an Excel file using `pandas`
- Displays the dataset as a table in the web app
